Гвидо Ван Россум

Guido van Rossum - род. 31 января 1956, Харлем, Нидерланды — голландский программист.

ВикипедиЯ
Python



Язык Python разработал голландский программист Гвидо Ван Россум (Guido van Rossum) в 1991 году.

Гвидо был фанатом британского комедийного сериала «Monty Python’s Flying Circus», откуда и пришло название языка.




Компания TIOBE Software опубликовала январский (09.01.2022 г.) рейтинг популярности языков программирования, в котором по сравнению с январём 2021 года выделяется перемещение языка Python с третьего на первое место.

Python подходит для решения широкого спектра проблем задач:

1. Системное программирование;
2. Графические приложения;
3. Веб-приложения;
4. Веб-сценарии;
5. Интеграция компонентов;
6. Приложения баз данных;
